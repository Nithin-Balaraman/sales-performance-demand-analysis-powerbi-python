# Project Documentation

This folder contains supporting documentation for the Sales Performance and Demand Analysis project.

## Files Included

| File Name            | Purpose                                                                                                    |
| -------------------- | ---------------------------------------------------------------------------------------------------------- |
| project_summary.md   | Provides the overall project objective, dataset overview, tools used, analysis performed, and business use |
| business_insights.md | Summarizes the key insights and business recommendations from the analysis                                 |
| data_dictionary.md   | Explains the columns used in the dataset and dashboard development                                         |

## Documentation Purpose

The purpose of this folder is to help recruiters, reviewers, and project evaluators understand the business problem, dataset structure, analysis process, and final insights without opening the Power BI file directly.
