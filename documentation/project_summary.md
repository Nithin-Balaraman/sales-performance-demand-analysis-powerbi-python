# Project Summary

## Project Title

Sales Performance and Demand Pattern Analysis using Power BI and Python

## Project Objective

The objective of this project is to analyze historical sales data and identify sales trends, product performance, regional performance, dealer contribution, and demand patterns. The project helps convert raw sales transactions into meaningful business insights for better sales planning, product strategy, dealer management, and demand analysis.

## Dataset Overview

- Dataset Type: Secondary sales transaction data
- Records Analyzed: 4.5 lakh+ sales records
- Time Period: FY 2019–2020 and FY 2020–2021
- Final Columns Used: 27
- Main Focus Areas: Sales performance, product category analysis, regional sales, dealer contribution, and demand classification

## Tools Used

- Microsoft Excel
- Power BI
- Power Query
- DAX
- Python
- Pandas
- SciPy
- Scikit-learn

## Analysis Performed

- Overall sales performance analysis
- Product-wise sales analysis
- Branch and region-wise sales analysis
- Dealer contribution analysis
- Monthly and quarterly trend analysis
- Demand classification analysis
- Correlation analysis
- One-way ANOVA
- Linear regression
- Independent sample t-test
- Coefficient of variation

## Business Use

This project helps identify which products, regions, and dealers contribute most to revenue. It also supports better decision-making in inventory planning, campaign timing, dealer relationship management, and product performance review.
