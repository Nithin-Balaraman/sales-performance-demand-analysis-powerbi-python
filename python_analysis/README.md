# Python Statistical Analysis

This folder contains the Python-based statistical analysis summary used in the Sales Performance and Demand Analysis project.

Python was used to validate the dashboard findings and support the business insights with statistical techniques.

## Statistical Techniques Used

### 1. Correlation Analysis

Used to understand the relationship between quantity sold and sales value.

### 2. One-Way ANOVA

Used to check whether sales value differs significantly across product categories.

### 3. Linear Regression

Used to understand how quantity sold affects sales value.

### 4. Independent Sample T-Test

Used to compare average transaction revenue between FY 2019-2020 and FY 2020-2021.

### 5. Coefficient of Variation

Used to compare sales stability and volatility across product categories.

## Note

The original company dataset is not uploaded due to confidentiality. The statistical results summary is provided to show the analysis approach and interpretation.
