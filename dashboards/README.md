# Power BI Dashboard Screenshots

This folder contains the dashboard screenshots created for the Sales Performance and Demand Analysis project.

## Dashboards Included

### 1. Overall Sales Performance Dashboard

This dashboard provides a high-level view of total sales value, quantity sold, yearly performance, monthly trend, and overall business performance. It helps understand whether sales improved or declined across the selected period.

### 2. Product Performance Analysis Dashboard

This dashboard analyzes product category contribution, product variants, and item-level performance. It helps identify which products are major revenue contributors and which product categories require business attention.

### 3. Regional Sales Analysis Dashboard

This dashboard shows sales performance across branches, states, and regions. It helps compare regional contribution and identify stronger and weaker sales locations.

### 4. Dealer Analysis Dashboard

This dashboard analyzes dealer-wise sales contribution. It helps identify top-performing dealers and understand dependency on major dealer accounts.

### 5. Demand Pattern Analysis Dashboard

This dashboard focuses on monthly and quarterly sales movement. It helps identify seasonal demand patterns and supports better planning for campaigns and inventory.

### 6. Demand Classification Dashboard

This dashboard classifies product categories based on demand behavior. It helps separate high-demand, stable, declining, and low-performing categories for better business decision-making.
